Motivaciones

CPU (Procesador)


El "cerebro" del computador. Se responsabiliza de ejecutar procesos, establecer conexiones y realizar calculos para mantener el sistema en orden.

Para saber qué tipo de CPU escoger, hay que prestar atención a 2 variables:

- Número de nucleos: A mayor cantidad de nucleos, mayor será la cantidad de procesos que se podrán ejecutar a la vez.

- Frecuencia: A mayor frecuencia, mayor será la velocidad con la que se ejecute cada proceso.

Placa Madre


La “columna vertebral” de un ordenador. En la Placa Madre se conectan todos los componentes internos de un computador.

Para elegir una Placa Madre debes considerar:

- Que el Chipset sea compatible con tu CPU

- Que las ranuras de memoria RAM sean compatibles con tus memorias RAM

- La cantidad máxima de discos duros que admite

- Que su tamaño sea el adecuado para tu gabinete

...etc

GPU (Tarjeta Gráfica)


La GPU se encarga de transformar todos los datos que le envía la CPU en información visual. Gracias a ellas podemos ver todas las imágenes que nos muestra el monitor de nuestro ordenador.

Al igual que una CPU, la GPU será más potente mientras mayor sea su frecuencia y número de núcleos.

Cuando la CPU no es lo suficientemente potente para seguir el ritmo de trabajo de la GPU (o viceversa), se produce un cuello de botella; el rendimiento del ordenador termina estando muy por debajo de lo que sus componentes prometen.

Para saber si tu CPU y GPU generan cuello de botella, puedes utilizar esta herramienta.
